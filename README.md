# Arduino-ultrasonik-seven-segment
## Materials
1. Arduino Uno
2. Jumper wire male to male
3. Mini breadboard
4. Ultrasonic HC-SR04
5. 4 digit 7 segment LED display
## Software
Arduino IDE https://www.arduino.cc/en/software
